# Galería Interactiva con JavaScript

Una galería de imágenes interactiva desarrollada con HTML, CSS y JavaScript.

## Características
- Agregar imágenes mediante URLs
- Seleccionar imágenes con un solo clic
- Eliminar imágenes seleccionadas
- Interfaz responsive
- Atajos de teclado
- Animaciones CSS

## Cómo usar
1. Abre `index.html` en tu navegador
2. Ingresa una URL de imagen en el campo
3. Haz clic en "Agregar Imagen" o presiona Enter
4. Haz clic en cualquier imagen para seleccionarla
5. Usa "Eliminar Imagen Seleccionada" para remover imágenes

## Tecnologías
- HTML5
- CSS3 (Grid, Flexbox, animaciones)
- JavaScript (manipulación del DOM)
- Font Awesome icons
- Google Fonts
